# ponto-eletronico 2

## Endpoints

### Autenticar
```bash
curl --location 'http://localhost:8080/autenticar' \
--header 'Content-Type: application/json' \
--data '{
    "matricula": "123456789",
    "senha": "fiap123"
}'
```

### Registrar ponto
```bash
curl --request POST \
  --url 'http://localhost:8080/registrar-ponto?matricula=123456789' \
  --header 'Content-Type: application/json'
```
